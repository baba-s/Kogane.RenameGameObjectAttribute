# UniRenameGameObjectAttribute

コンポーネントをアタッチした時にゲームオブジェクトの名前をコンポーネント名にリネームするエディタ拡張

## 使用例

```cs
using Kogane;
using UnityEngine;

[RenameGameObject]
public sealed class Example : MonoBehaviour
{
}
```
